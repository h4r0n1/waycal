waycal

a tiny calendar popup for waybar

click a calendar icon in your bar, the month drops down, arrow keys navigate, escape closes. that's it.

Install

Arch · Omarchy yay -S waycal click to copy
Any distro w/ Rust cargo install waycal click to copy
Prebuilt binary curl … | tar -xz click to copy

Controls

← →
previous / next month
↑ ↓
previous / next year
⏎
jump back to today
⎋
close
